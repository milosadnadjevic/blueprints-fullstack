
## Setup Instructions

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

Backend runs on `http://localhost:5001`

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on `http://localhost:3000`
